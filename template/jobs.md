# Jobs

Tracker for saved jobs and applications. One row per role. Flip `Status` as it
moves through the funnel. (Optional: this same table can serve as a Texas TWC
work-search log, which requires at least 3 activities per week.)

Status values: `saved` / `applied` / `screen` / `interview` / `offer` / `rejected` / `withdrawn` / `ghosted`

Activity values: `applied online` / `interview` / `job fair` / `networking` / `workshop` / `staffing agency`

For anything `applied` or later, create `applications/<company>-<role>/` with `job.md`, `resume.md`, `cover-letter.md`.
