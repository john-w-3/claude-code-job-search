# Sharing this guide

This folder is built to become its own small public project: a `README.md` that
reads as a step-by-step guide, plus a `template/` starter kit people can copy to
begin their own search.

## Important: this is the sanitized, shareable copy

This folder contains no real personal data, on purpose. Your actual job-search
folder holds real names, salary numbers, and interview notes, and it should stay
in a **private** repo. Only this generic version is safe to make public. Do not
publish your real search.

## Option A: a public GitHub repo (simplest, one link to share)

1. Create a **new, public** repository on GitHub (for example
   `claude-job-search`). Do not reuse your private job repo.
2. Copy the contents of this `share/` folder into it.
3. Commit and push. The `README.md` becomes the landing page automatically, so the
   link to the repo is already a readable guide.
4. (Optional) In the repo's Settings, check **Template repository**. Then anyone
   can click **Use this template** to start their own copy of the starter kit with
   one button.

If you are not sure how to do any of this, ask Claude Code itself: "create a new
public GitHub repo from this folder and push it." It will walk you through it.

## Option B: no GitHub at all

Everything here is plain Markdown and a folder of template files. You can zip the
folder and email it, drop it in a shared drive, or hand someone the `README.md` on
its own. It needs no server and no build step. The person you share it with just
needs to install Claude Code and copy the `template/` folder to get started.
